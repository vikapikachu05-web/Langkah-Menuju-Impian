# Trajectory.io
Trajectory adalah jalur objek menuju sasaran.

Di proyek ini, lintasan itu adalah simbol perjalanan mimpimu. Aplikasi ini mengubah target abstrak menjadi koordinat yang jelas: mulai dari Titik Awal (Latar Belakang), Navigasi (Langkah), hingga mendarat di Target Akhir (Kesimpulan).

Setiap langkah dipetakan agar mimpimu tidak meleset.
